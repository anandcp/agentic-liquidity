# Data Folder

This folder contains sample datasets used in the paper and for code verification.

### Files

- **sample_repo_panel.csv**  
  A small synthetic repo–month panel generated for demonstration and testing.

- **liquidity_efficiency_curve.csv**  
  ASCII data used to plot the inverted-U liquidity–efficiency curve in the paper.

### Notes

The full panel used in the paper should be generated via: 
<code> python analysis/generate_repo_panel.py </code>

These sample files allow quick testing without regenerating large datasets.
